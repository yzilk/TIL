★更新の実行
1. DB に接続し、Connection のオブジェクトを取得する。
2. 実行したい SQL を String 型の文字列として定義する。
3. PreparedStatement のオブジェクト（ステートメント)を生成する。
4. SQL を実行する

update→戻り値は処理件数
（対してexeQuery→result）
